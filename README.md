# authentication
  For Development--->
  
     1. npm i     (To get modules)
     2. set config variables
             > auth_jwtPrivateKey="yourSecretKey"
             > auth_db="yourDbUrl"
             > auth_mailPass="123@#abc"                (to send mail)
  
  For Production--->           
  
     1. connect to github and set repo
     2. set config variables
             > auth_jwtPrivateKey="yourSecretKey"
             > auth_db="yourDbUrl"
             > auth_mailPass="123@#abc"                (to send mail)
